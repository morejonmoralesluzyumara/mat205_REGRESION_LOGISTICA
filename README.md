# mat205_REGRESION_LOGISTICA
# Predicción de Abandono de Clientes Bancarios mediante Regresión Logística

## Descripción

Este proyecto implementa un modelo de Regresión Logística para predecir si un cliente abandonará o no un banco utilizando variables demográficas y financieras.

## Dataset

Bank Customer Churn Dataset obtenido de Kaggle.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Instalación

```bash
pip install pandas numpy matplotlib scikit-learn
```

## Ejecución

1. Abrir el notebook `regresion.ipynb`.
2. Agregar el customer-churn-records.csv
3. Ejecutar las celdas en orden.
4. Analizar los resultados obtenidos.

## Resultados

El modelo alcanzó una precisión aproximada del 81.1% en el conjunto de prueba después de eliminar variables con correlación excesiva.
